# MMM_MyWineApp

## Projet “Cave à vin”Réaliser une application de gestion de sa cave à vin. 

  ### Fonctionnalités:

      -accès/stockage des données (Région, Appelation, Année, Nom, Photo)

      -maturité des vins-cartes des vins (cf infos sur le site techo-resto.org)

      -reconnaissance du vin par la photo (via l'API Kooaba de reconnaissance d'images)
